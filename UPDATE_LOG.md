# Snipee アップデート履歴

---

## Version 1.3.0 (2025-11-20)

### 🎯 新機能

- **スニペット専用ホームの UI 改善**: 簡易ホームと同じインラインサブメニュー構造に統一
  - 別ウィンドウではなく、同一 HTML 内にサブメニューを表示
  - キーボード操作が 1 つのファイル内で完結
  - フォーカス問題を解決

### 🔧 改善

- **表示順の変更**: マスタスニペット → 個別スニペットの順に統一
  - 簡易ホーム、スニペット専用ホームともに同じ順序
- **フォントサイズの統一**: フォルダ名・スニペット名を 11px に調整
- **itemHeight の最適化**: メインアイテム・サブメニューアイテムの高さを調整
  - メインアイテム: padding 3px
  - サブメニューアイテム: padding 3px、itemHeight 22px
- **ウィンドウサイズの変更**:
  - 幅: 460px（メイン 210px + サブメニュー 250px）
  - 高さ: 650px（より多くのスニペットを表示可能に）
- **サブメニューの表示位置**: 常にウィンドウ上部から表示するように変更
  - スニペット数が多い場合のみスクロール可能
  - 見切れる問題を解決

### 🐛 バグ修正

- Electron での複数ウィンドウ間 IPC 通信によるフリーズ問題を修正
  - submenu.html を削除し、インラインサブメニューに統一
- サブメニューの表示位置計算の問題を修正

### ⚠️ 重要な設計変更

- **アーキテクチャの改善**:
  - 複数ウィンドウ間の IPC 通信を廃止
  - サブメニューをインライン表示に統一
  - これにより安定性が大幅に向上

## Version 1.2.0 (2025-11-19)

### 🎯 新機能

- **個別スニペット機能**: ローカル保存の個別スニペットを追加・編集・削除できる専用ウィンドウを実装
  - フォルダ分類機能（追加・削除可能）
  - スニペットの CRUD 機能（作成・編集・削除）
  - 簡易ホームに「✏️ スニペット編集」メニューを追加
  - 個別スニペットとマスタスニペットを統合表示（個別 → マスタの順）
  - フォルダ階層表示（折りたたみ可能）
  - 自動保存機能

### 📝 仕様

- **個別スニペット**: ローカル保存、編集可能、個人用
- **マスタスニペット**: Google Drive 連携、読み取り専用、チーム共有

### 🔧 改善

- `snippet-editor.html` を Clipy ライクなシンプルデザインに全面リニューアル
  - Apple 風の洗練された UI
  - タイトルと内容のみのシンプルな編集エリア
  - フォルダ選択を削除（フォルダの下に直接スニペットを配置）
- `settings.html` のホットキー変更モーダルを改善
  - 背景クリックで閉じる機能を追加
  - Esc キーで閉じる機能を追加
- フォントサイズを 13px に統一

### 🐛 バグ修正

- スニペット専用ホーム（Option+Shift+V）で透明な巨大ウィンドウが表示される問題を修正
  - `main.js` の `createSnippetWindow()` の width を 420px → 210px に修正
  - `positionAndShowSnippet()` のウィンドウ位置計算を修正
- ホットキー変更時に「CC」と表示される問題を修正
  - Modifier キー単体での入力を無効化
- 環境設定のホットキー変更モーダルが閉じずに画面操作できなくなる問題を修正
  - イベントリスナーを `DOMContentLoaded` 内に正しく配置

## Version 1.1.0 (2025-11-11)

### 🎯 新機能

- **ウィンドウ位置の記憶**: ウィンドウを移動した位置を記憶し、次回起動時に同じ場所に表示できるように（環境設定で切り替え可能）
- **クリップボード履歴のピン留め**: よく使う履歴項目をピン留めして、常に最上部の「● ピン留め」グループに表示できるように

### 🔧 改善

- スニペット専用ホームの表示位置を調整（カーソル位置により近く）
- サブメニューの幅を 160px→200px に拡大
- サブメニューのテキストを 1 行で表示（25 文字以上は省略）
- 起動時のスニペット読み込みログを削除（ターミナル出力をクリーンに）
- サブメニューの高さを可変にして全項目を表示可能に
- アイコンを変更（履歴: 📄、フォルダ: 📁）
- フッターを 2 段表示に変更（操作方法を見やすく）
- フォントサイズを統一（13px）
- アクションアイテムのフォントサイズを 12px に調整
- サブメニューのスクロール速度を改善（smooth→auto）
- ウィンドウの黒枠を削除（hasShadow: false）
- サブメニューのスクロールバーをカスタマイズ（4px 幅、クリーム色）
- 環境設定でヘッダーのみドラッグ可能にしてスクロールとの両立を実現

### 🐛 バグ修正

- 履歴データの型情報が失われていた問題を修正
- ピン留めボタンがクリックできなかった問題を修正
- デバッグログを全削除（本番環境用に最適化）

- ホットキーカスタマイズ（簡易ホーム: Alt+Shift+C、スニペット専用: Alt+Shift+V）
- フォルダの表示/非表示設定
- Mac 権限管理（アクセシビリティ）
- クロスプラットフォーム対応（Mac/Windows）
- 自動ペースト機能
