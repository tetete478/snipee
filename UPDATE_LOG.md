# Snipee 更新履歴

---

## Version 1.5.5 (2025-11-27)

### 🎉 正式リリース！

Snipee がついに正式リリースとなりました！チーム全員でお使いいただけます。

### ✨ 新機能

- **マスタ編集モード**:

  - パスワード（1108）を入力すると、マスタスニペットを編集できるようになります
  - 編集後は XML エクスポートで Google Drive に保存できます

- **XML エクスポート機能**:

  - スニペットを Clipy 互換の XML 形式でエクスポートできます
  - エクスポートするフォルダをチェックボックスで選択可能

- **変数機能**:

  - スニペット内に変数を記述すると、貼り付け時に自動で置換されます
  - `{今日:MM/DD}` → 今日の日付（例: 11/27）
  - `{明日:MM/DD}` → 明日の日付
  - `{タイムスタンプ}` → 年月日＋時分秒
  - `{名前}` → 設定画面で登録した名前

- **Mac 自動ペースト完成**:

  - Bundle ID 方式で元のアプリに確実に戻ってペーストします
  - アクセシビリティ権限を許可してお使いください

- **マニュアル追加**:
  - 設定画面の「一般」タブからマニュアルにアクセスできます
  - Mac/Windows 両対応の完全版マニュアル 4 部作

### 🎨 UI 改善

- **枠線追加**:

  - 簡易ホーム・サブメニューに枠線を追加して視認性を向上しました
  - 白い背景でもウィンドウの境界がはっきり見えます

- **スクロールバー統一**:

  - 全てのスクロールバーを 4px に統一してスッキリした見た目に

- **スニペット編集画面改善**:

  - フォントサイズを 12px に調整
  - 説明パネルの幅を拡大（160px → 190px）

- **カラー調整**:
  - グレー系カラーを濃くして視認性向上

### 🐛 バグ修正

- **Google Drive 同期の改善**:

  - 5 分ごとの定期同期を削除（起動時のみに変更）
  - ローカルで編集した内容が上書きされる問題を解決

- **Backspace バグ修正**:

  - テキストエリアで Backspace が正常に動作するようになりました

- **リサイズ改善**:
  - 選択移動時のウィンドウのガタつきを解消
