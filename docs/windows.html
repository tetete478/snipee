1. Snipeeとは？

Snipee（スナイピー）は、仕事を効率化するためのツールです。

🎯 Snipeeでできること

機能説明
📋クリップボード履歴
過去にコピーした内容を100件まで保存。いつでも呼び出せます。
📝スニペット（定型文）
よく使う文章を登録しておいて、ワンクリックで貼り付け。
☁️チーム共有（マスタスニペット）
Google Driveを使って、チーム全員で同じスニペットを共有。
🔄自動アップデート
新しいバージョンがあれば自動で通知。再起動するだけ。
毎回「お世話になっております」と打つ手間から解放されます！

2. インストール方法

Snipeeをパソコンにインストールしましょう。

📥 インストール手順

  1. 以下のURLから「Snipee Setup.exe」をダウンロード
    https://drive.google.com/file/d/1J7gFnD3mzgCmI2eq_vYn2avT5ilfYzJg/view
    ダウンロード先：デスクトップでもOK
  2. ダウンロードした「Snipee Setup.exe」をダブルクリック
  3. インストールが自動で始まります（数秒で完了）
  4. インストール完了！Snipeeが自動で起動します
「WindowsによってPCが保護されました」と表示されたら「詳細情報」→「実行」をクリック
画面右下のタスクトレイに「📋」アイコンが表示されたら成功です！

💡 タスクトレイとは？

画面右下の時計の横にあるアイコンが並んでいる場所です。Snipeeは常にここに常駐しています。

アイコンが見当たらない場合は「∧」（上向き矢印）をクリックすると隠れているアイコンが表示されます

3. 初期設定

チームで共有しているスニペットを使えるように設定しましょう。

⚙️ 設定手順

  1. タスクトレイの「📋」アイコンを右クリック
  2. メニューから「環境設定」をクリック
  3. 「一般」タブで自分の名前を入力【必須】
    → スニペット内の「{名前}」が自動で置き換わるようになります
    → 例：「{名前}です。」→「山田です。」
  4. 「同期」タブをクリック
  5. 「Google Drive URL」に以下のURLを貼り付け
    https://drive.google.com/file/d/1MIHYx_GUjfqv591h6rzIbcxm_FQZwAXY/view
  6. 「保存」ボタンをクリック
  7. 「今すぐ同期」ボタンをクリック
「同期成功」と表示されたら完了です！

4. 基本の使い方

4-1. 簡易ホーム（全体メニュー）

Snipeeの全機能にアクセスできるメインメニューです。最もよく使う画面なので、まずはこれを覚えましょう！

🔑 呼び出し方

Ctrl ＋ Alt ＋ C

または、タスクトレイの「📋」アイコンをクリックでも開けます。

📋 簡易ホームでできること

項目説明
📋 クリップボード履歴最近コピーした内容が表示されます。クリックで貼り付け。
📁 スニペットフォルダ登録済みのスニペットにアクセス。フォルダをクリックで展開。
🗑️ 履歴をクリアクリップボード履歴を一括削除（ピン留めは残ります）
⚙️ 環境設定各種設定画面を開く
✏️ スニペット編集個別スニペットの追加・編集画面を開く

💡 使い方のコツ

  • 上下キー（↑↓）で項目を選択、Enterキーで実行
  • フォルダは右矢印キー（→）で展開、左矢印キー（←）で閉じる
  • ESCキーで閉じる
迷ったらとりあえず Ctrl+Alt+C！すべての機能にアクセスできます

4-2. スニペット簡易ホーム

スニペット（定型文）だけを表示するシンプルな画面です。スニペットをすぐに使いたいときはこちらが便利！

🔑 呼び出し方

Ctrl ＋ Alt ＋ V

📝 スニペットの使い方

  1. フォルダをクリックして開く
  2. 使いたいスニペットをクリック
  3. 自動で貼り付けされます！

🎨 スニペットの種類

🔒 マスタチーム全員で共有しているスニペット（変更不可）
📄 個別自分だけのスニペット（自由に追加・編集可能）
スニペットだけ使いたいなら Ctrl+Alt+V が最速！

4-3. 履歴ホーム

クリップボード履歴だけを表示する専用画面です。過去にコピーした内容を素早く呼び出せます。

🔑 呼び出し方

Ctrl ＋ Alt ＋ X

📋 履歴の使い方

  1. 履歴一覧から貼り付けたい項目をクリック
  2. 自動で貼り付けされます！
キーボードの上下キー（↑↓）で選択して、Enterキーで貼り付けもできます

📌 ピン留め機能

よく使う履歴は「ピン留め」できます。

  • 項目にカーソルを合わせる → 右側に「○」が表示される
  • 「○」をクリックすると「●」に変わり、ピン留め完了
  • ピン留めした項目は削除されず、一覧の上部に固定されます

📊 3つのホットキーまとめ

Ctrl+Alt+C Ctrl+Alt+V Ctrl+Alt+X
簡易ホーム スニペット 履歴
全部入り 定型文だけ コピー履歴だけ

5. マスタスニペットについて

⭐ マスタスニペットとは

小松が普段から使っている「最強のメッセ対応テンプレート」です。チーム全員で同じスニペットを使えるよう、Google Driveで共有しています。

重要：使う前に必ず「説明」を読んでください！

マスタスニペットは小松が作成したものです。各スニペットには「どんな場面で使うか」「注意点」などの説明があります。間違った使い方を防ぐため、必ず説明を確認してから使いましょう。

📖 説明の確認方法

  1. 簡易ホーム（Ctrl+Alt+C）または スニペット編集 を開く
  2. マスタフォルダ内のスニペットをクリック
  3. 右側に「説明」と「内容」が表示される
  4. 説明をよく読んでから使用する

✨ マスタスニペットの特徴

📱 スマホ最適化スマホ閲覧に最適な改行設定済み。お客様がスマホで見ても読みやすい！
🎯 場面対応挨拶・お礼・謝罪・案内など、多数の場面に対応したテンプレートを用意
🔄 自動更新小松がテンプレを更新すると、全員のSnipeeに自動で反映
⚡ 変数対応「{名前}」「{今日}」などが自動で置き換わる

📁 フォルダ構成の例

  • 挨拶：お世話になっております、はじめまして、など
  • お礼：ご連絡ありがとうございます、お問い合わせありがとうございます、など
  • 案内：お支払い方法、配送について、など
  • 謝罪：お待たせして申し訳ございません、など
  • 締め：よろしくお願いいたします、など
フォルダ構成は小松が随時更新します。新しいテンプレが追加されることも！

🔒 マスタスニペットは変更できません

マスタスニペットはチーム共有のため、個人での変更はできません。これは、チーム全員が同じ高品質なテンプレートを使うための仕組みです。

自分用にアレンジしたい場合は、個別スニペットとして作成できます。詳しくは次の「6. スニペット編集画面の使い方」をご覧ください。

6. スニペット編集画面の使い方

自分だけの「個別スニペット」を追加・編集できる画面です。

📝 開き方

  • 簡易ホーム（Ctrl+Alt+C）→ 「スニペット編集」をクリック
  • または、タスクトレイ右クリック → 「スニペット編集」

🖥️ 画面の構成

場所内容
左側フォルダ一覧（マスタ / 個別）
右側選択したスニペットの詳細（タイトル・説明・内容）

➕ 新しいスニペットを追加する

  1. 左側の「個別」フォルダをクリックして選択
  2. 画面上部の「＋スニペット追加」ボタンをクリック
  3. 右側に入力欄が表示される
  4. 「タイトル」「説明」「内容」を入力
  5. 「保存」ボタンをクリック
これで新しいスニペットが追加されました！

✏️ 既存のスニペットを編集する

  1. 編集したいスニペットをクリック
  2. 右側の内容を修正
  3. 「保存」ボタンをクリック
マスタスニペットは編集できません（閲覧のみ）

🗑️ スニペットを削除する

  1. 削除したいスニペットをクリック
  2. 「削除」ボタンをクリック
  3. 確認メッセージで「OK」をクリック
削除したスニペットは元に戻せません。ご注意ください。

📁 フォルダを追加する

スニペットを整理するためのフォルダを追加できます。

  • 「個別」を右クリック → 「フォルダ追加」
  • フォルダ名を入力 → 「OK」

💡 マスタスニペットを参考にする

マスタスニペットをベースに自分用のスニペットを作る方法：

  1. マスタスニペットをクリックして内容を確認
  2. 「内容」欄のテキストをコピー（Ctrl+C）
  3. 個別フォルダで「＋スニペット追加」
  4. コピーした内容を貼り付けてアレンジ
  5. 「保存」で完成！
変数（{名前}など）も自分のスニペットで使えます！

7. 便利な変数機能

スニペット内に特別な文字を入れると、貼り付け時に自動で置き換わります。

📅 使える変数一覧

スニペットに書く内容貼り付け時の結果
{名前}設定した名前（例：山田）
{今日:MM/DD}今日の日付（例：12/07）
{明日:MM/DD}明日の日付（例：12/08）
{今日:M月D日}今日の日付（例：12月7日）
{タイムスタンプ}例：2024/12/07 14:30:45

💡 使用例

スニペットに登録する内容：

お世話になっております。{名前}です。
{明日:M月D日}までにご連絡いただけますと幸いです。

貼り付け時に置き換わった結果：

お世話になっております。山田です。
12月8日までにご連絡いただけますと幸いです。

9. 困ったときは

❓ よくある質問

Q1. ショートカットキーが効かない

  • 日本語入力（IME）がONになっていませんか？→ 半角英数モードにしてください
  • タスクトレイにSnipeeのアイコンがありますか？→ なければ再起動してください
  • 他のアプリとショートカットが競合していませんか？→ 環境設定で変更できます

Q2. スニペットが表示されない

  • 環境設定 → 「同期」タブ → 「今すぐ同期」をクリック
  • Google Drive URLが正しく設定されているか確認
  • インターネット接続を確認

Q3. 貼り付けされない

  • 貼り付け先のアプリがアクティブ（選択されている）か確認
  • カーソルが入力欄にあるか確認
  • もう一度試してみてください

Q4. {名前}が置き換わらない

環境設定 → 「一般」タブで名前が登録されているか確認してください。

Q5. アップデートの通知が来た

「再起動」ボタンをクリックするだけでOK！自動で最新版になります。

Q6. 特定のフォルダを非表示にしたい

環境設定 → 「表示」タブで、使わないフォルダのチェックを外してください。

📞 それでも解決しない場合

小松またはIT担当者にご連絡ください。
