# Clipy Windows - Mac開発者向け補足

## ⚠️ Mac環境での自動ペーストについて

macOSのセキュリティ制限により、**自動ペースト機能はWindowsのみ有効**です。

### Mac環境での動作

1. `Command + Shift + V` でクリップボード履歴を開く
2. ↑↓キーで選択
3. **Enter** で選択項目をクリップボードにコピー
4. **手動で `Command + V`** でペースト

### Windows環境での動作

1. `Ctrl + Shift + V` でクリップボード履歴を開く
2. ↑↓キーで選択
3. **Enter** で選択項目を自動ペースト（`Ctrl + V`が自動実行）

## 🎯 これは仕様です

- Macでは自動ペーストにAppleScriptの実行権限が必要
- 権限設定が複雑でユーザーに負担をかけるため
- シンプルに「Enterでコピー → 手動ペースト」方式を採用

## 💡 実際の使用フロー（Mac）

```
1. テキストをコピー（Command + C）
   → 自動で履歴に記録

2. Command + Shift + V を押す
   → 履歴ウィンドウが開く

3. ↑↓キーで選択、または検索
   
4. Enter を押す
   → クリップボードにコピーされる
   → ウィンドウが閉じる

5. Command + V でペースト
   → 選択したテキストが貼り付けられる
```

慣れると超高速です! ⚡️

## 🔧 なぜMacで自動ペーストが難しいか

### 必要な権限
- **アクセシビリティ権限**の付与が必要
- ユーザーが「システム設定 → プライバシーとセキュリティ → アクセシビリティ」で手動許可

### 問題点
- 初回起動時に複雑な権限設定が必要
- 権限がないとエラーが出る（今回のエラー）
- ユーザー体験が悪い

### 解決策
- **シンプルな方式を採用**
- Enterでコピー、手動でペースト
- Clipy本家も同じ方式です

## ✅ 結論

**これは不具合ではなく、意図的な設計です**

Mac環境:
- Enter → コピー
- Command + V → ペースト（手動）

Windows環境:
- Enter → 自動ペースト

両環境で快適に使えます! 🎉

---

**追記:** 
もしMacでも自動ペーストが必要な場合は、アプリにアクセシビリティ権限を付与する必要がありますが、チーム利用を考えると現在の方式が最適です。
