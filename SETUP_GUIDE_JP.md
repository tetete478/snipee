# Clipy Windows - セットアップガイド（Windowsユーザー向け）

## 📦 インストール方法

### ステップ1: インストーラーを実行

1. `ClipyWindows Setup 2.0.0.exe` をダブルクリック
2. インストール先を選択（デフォルトでOK）
3. 「インストール」をクリック
4. 完了したら「完了」をクリック

これで完了です!

### ステップ2: 初回設定

アプリを起動すると設定画面が開きます。

## 🎯 基本的な使い方

### クリップボード履歴を使う

1. **普通にテキストをコピー**
   - 通常通り `Ctrl + C` でコピー
   - 自動的に履歴に保存されます

2. **履歴から貼り付け**
   - `Ctrl + Shift + V` を押す
   - 履歴一覧が表示される
   - ↑↓キーで選択
   - Enterで貼り付け

### ショートカットキー一覧

| 操作 | キー |
|------|------|
| 履歴を開く | `Ctrl + Shift + V` |
| 選択を移動 | `↑` / `↓` |
| 貼り付け | `Enter` |
| ウィンドウを閉じる | `Esc` |
| タブ切り替え | `Tab` |
| 検索 | 文字を入力するだけ |

## 📝 共有スニペットを使う

管理者から共有されたスニペット（定型文）を使えます。

### 設定方法

1. システムトレイのアイコンを右クリック → 「設定」
2. 「共有スニペット設定」に管理者から受け取ったURLを入力
3. 「保存して同期」をクリック

これでチーム全員が同じスニペットを使えます!

### スニペットを使う

1. `Ctrl + Shift + V` で履歴を開く
2. 「共有」タブをクリック
3. 使いたいスニペットを選択
4. Enterで貼り付け

## ⭐ 個人用スニペットを作る

よく使う文章を登録できます。

1. 設定画面を開く
2. 「個人スニペット」→「新規スニペット追加」
3. タイトルと内容を入力
4. 保存

例:
- メールの署名
- よく使う挨拶文
- コードのテンプレート

## 💡 よくある質問

### Q: 履歴はいつまで保存される?

A: 最大100件まで保存されます。古いものから自動削除されます。

### Q: アプリを終了したい

A: システムトレイのアイコンを右クリック → 「終了」

### Q: 履歴を削除したい

A: 設定画面 → 「履歴を全削除」

### Q: 自動起動したい

A: スタートアップに登録してください:
1. `Win + R` で「shell:startup」を実行
2. ClipyWindowsのショートカットをここに配置

### Q: ショートカットキーが効かない

A: 
- アプリが起動しているか確認（システムトレイにアイコンがある）
- 他のアプリと競合している可能性があります
- アプリを再起動してみてください

### Q: 画像はコピーできる?

A: 現在はテキストのみ対応しています。

## 🔒 セキュリティについて

- 履歴はすべてローカルに保存されます
- インターネットに送信されることはありません
- 共有スニペットのみGoogle Drive経由で取得します
